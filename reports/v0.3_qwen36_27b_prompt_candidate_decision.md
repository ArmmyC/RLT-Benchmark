# v0.3 qwen36-27b Prompt Candidate Decision

This report summarizes the v0.3 prompt-profile smoke matrix for `qwen36-27b` and selects candidates for larger validation runs.

The smoke matrix is intentionally small: 3 samples per benchmark and prompt profile, across VerilogEval, RTLLM, and RTL-OPT. These results are useful for screening obviously weak candidates, but they are not enough to claim final prompt superiority.

## Decision

Select these prompt profiles for larger validation:

- `neutral_baseline`
- `strict_rtl_only`

Hold back these prompt profiles for now:

- `low_area_low_power`
- `no_explanation_code_only`

The selected profiles tied for the best smoke total at 7/9 passed samples and had no compile failures. `low_area_low_power` remained competitive on VerilogEval and RTLLM but was weaker on RTL-OPT equivalence smoke. `no_explanation_code_only` had the weakest total and one RTLLM compile failure.

## Observations

All four profiles passed the 3-sample VerilogEval smoke set, so VerilogEval alone does not separate the candidates.

RTLLM separated `no_explanation_code_only` from the rest: it passed 1/3, while `neutral_baseline`, `strict_rtl_only`, and `low_area_low_power` each passed 2/3. It also had one compile failure, which matters because v0.3 is partly testing extraction and formatting reliability.

RTL-OPT equivalence separated the larger-run candidates more clearly. `neutral_baseline` and `strict_rtl_only` each passed 2/3. `low_area_low_power` and `no_explanation_code_only` each passed 1/3, both with two equivalence failures.

The low-area/low-power prompt should not be treated as a PPA result. The smoke runs did not establish area, power, or timing improvement through synthesis reports. Its current value is exploratory, not a default candidate.

## Limitations

This report does not claim that one prompt is finally better than another. It only selects larger-run candidates based on a small smoke-screening matrix.

This report does not claim final silicon PPA quality. Area and power claims require synthesis/PPA evidence, not prompt intent alone.
