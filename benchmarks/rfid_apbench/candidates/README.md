# RFID-APBench Public Candidate Fixtures

This directory contains public/synthetic candidate RTL fixtures for exercising the candidate evaluator before any model jobs are run.

`reference_copy/` intentionally mirrors the committed public reference RTL for each task. It is a smoke-test candidate set, not model-generated output and not a benchmark result.

Do not commit raw model responses, model-generated RTL, private RTL, VCD files, simulator logs, synthesis logs, compiled artifacts, model weights, training datasets, or adapter checkpoints here.
